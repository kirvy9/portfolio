
# QA 자동화 테스트 포트폴리오

## 📌 개요
본 저장소는 국비교육 과정과 개인 학습을 통해 수행한 QA 테스트 프로젝트들을 정리한 포트폴리오입니다.  
웹/모바일 환경에서의 기능 테스트, 자동화 테스트, 성능/접근성 분석 등을 다양한 도구로 실습하였습니다.
